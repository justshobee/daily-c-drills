# camel_to_snake

**Assignment name :** camel_to_snake  
**Expected files :** camel_to_snake.c  
**Allowed functions:** write

---

Write a program that converts CamelCase (first arg) to snake_case and prints lowercased snake_case followed by newline.

```
$> ./camel_to_snake 'HelloWorld' | cat -e
hello_world$
```
