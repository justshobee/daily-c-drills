# is_pangram

**Assignment name :** is_pangram  
**Expected files :** is_pangram.c  
**Allowed functions:** write

---

Write a program that prints '1' and newline if the first argument contains every letter of the alphabet at least once (ignore case), otherwise prints '0' and newline.

```
$> ./is_pangram 'The quick brown fox jumps over the lazy dog' | cat -e
1$
```
