# palindrome_pairs

**Assignment name :** palindrome_pairs  
**Expected files :** palindrome_pairs.c  
**Allowed functions:** write

---

Write a program that given N then N words, prints pairs of indices i j such that concat(word_i, word_j) is a palindrome. Print each pair on a new line as 'i j'. If none, print newline.

```
$> ./palindrome_pairs 3 bat tab cat | cat -e
0 1$
1 0$
```
