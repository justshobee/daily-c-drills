# rle_encode

**Assignment name :** rle_encode  
**Expected files :** rle_encode.c  
**Allowed functions:** write

---

Write a program that performs run-length encoding on the first argument string (e.g., 'aaabb' -> 'a3b2') and prints result followed by newline.

```
$> ./rle_encode 'aaabbc' | cat -e
a3b2c1$
```
