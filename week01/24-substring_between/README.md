# substring_between

**Assignment name :** substring_between  
**Expected files :** substring_between.c  
**Allowed functions:** write

---

Write a program that extracts and prints the substring between the first occurrence of two characters given as second and third args. If not found, print newline.

```
$> ./substring_between 'a[b]c' '[' ']' | cat -e
b$
```
