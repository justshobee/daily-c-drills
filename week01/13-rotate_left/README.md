# rotate_left

**Assignment name :** rotate_left  
**Expected files :** rotate_left.c  
**Allowed functions:** write

---

Write a program that rotates the characters of the first argument left by N positions (second arg, non-negative integer) and prints the result followed by newline. If N is missing, use 1.

```
$> ./rotate_left 'abcdef' 2 | cat -e
cdefab$
```
