# atoi_write

**Assignment name :** atoi_write  
**Expected files :** atoi_write.c  
**Allowed functions:** write

---

Write a program that converts the first argument string to an integer (handle leading spaces and optional + or -) and prints the integer followed by newline. If no arg, print 0 and newline.

```
$> ./atoi_write ' -123abc' | cat -e
-123$
```
