# is_isogram

**Assignment name :** is_isogram  
**Expected files :** is_isogram.c  
**Allowed functions:** write

---

Write a program that prints "1" and newline if the first argument is an isogram (no repeating letters, ignore case), otherwise prints "0" and newline.

```
$> ./is_isogram 'Dermatoglyphics' | cat -e
1$
```
