# 06-count_substring_occurrences

**Assignment name:** count_substring_occurrences

**Expected files:** count_substring_occurrences.c

**Allowed functions:** write

## Description

Write a program that counts non-overlapping occurrences of the second argument in the first argument and prints the count followed by newline.

## Example

```
$> ./count_substring_occurrences 'abababa' 'aba' | cat -e
2$
```
