# 04-is_palindrome

## Assignment name
is_palindrome

## Expected files
is_palindrome.c

## Allowed functions
write

## Description
Write a program that prints "1" and a newline if the first argument is a palindrome (ignore spaces and case), otherwise prints "0" and newline. If no argument, print newline.

## Example
```
$> ./is_palindrome 'A man a plan a canal Panama' | cat -e
1$
```
