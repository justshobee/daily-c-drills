# replace_substring

**Assignment name :** replace_substring  
**Expected files :** replace_substring.c  
**Allowed functions:** write

---

Write a program that replaces all non-overlapping occurrences of the second arg in the first arg with the third arg and prints result followed by newline.

```
$> ./replace_substring 'foo bar foo' 'foo' 'baz' | cat -e
baz bar baz$
```
