# longest_palindromic_substring

**Assignment name :** longest_palindromic_substring  
**Expected files :** longest_palindromic_substring.c  
**Allowed functions:** write

---

Write a program that prints the longest palindromic substring of the first argument. If multiple with same length, print the first. End with newline.

```
$> ./longest_palindromic_substring 'babad' | cat -e
bab$
```
