# ./02-count_words.txt

**Assignment name:** count_words

**Expected files:** count_words.c

**Allowed functions:** write

Write a program that counts the number of words in the first argument string and prints the count followed by a newline. Words are separated by spaces or tabs. If no argument, print 0 and newline.

```
$> ./count_words "Hello world from C" | cat -e
4$
```
