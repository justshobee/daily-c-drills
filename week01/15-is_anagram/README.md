# is_anagram

**Assignment name :** is_anagram  
**Expected files :** is_anagram.c  
**Allowed functions:** write

---

Write a program that prints "1" and newline if the two arguments are anagrams of each other (ignore spaces and case), otherwise prints "0" and newline.

```
$> ./is_anagram "William Shakespeare" "I am a weakish speller" | cat -e
1$
```
