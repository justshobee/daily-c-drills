# daily-c-drills

**Daily-c-drills** is a dedicated space for mastering the C programming language through consistent, daily practice. The repository tracks a journey through algorithmic challenges and syntax mastery, primarily following the structured curriculum provided by **Portfolio Courses**.

##  Project Overview
The goal of this repository is to build a strong foundation in low-level programming by solving practical problems. Each exercise focuses on core concepts such as:

*   **Logic building:** Solving problems with loops, conditionals, and recursion.
*   **Memory Management:** Understanding pointers, arrays, and dynamic allocation.
*   **Data Structures:** Implementing fundamental structures in C.

##  Source Material
